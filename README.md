This program is a simple Intramurals Sports Tracker for a class section. It asks for the section name and the sports coordinator’s name, then shows a list of sports events. The user can enter up to 4 games by choosing a sport number, inputting the opponent section, and entering the result (W for win or L for loss). Each win gives 3 points and each loss gives 0 points. After all entries, the program calculates the total points and determines the section’s standing (GOLD CONTENDER, SILVER PUSH, or KEEP FIGHTING). Finally, it displays a complete game results board with all the details. To run the program, open it in a Python IDE or any code editor, then execute the file. Follow the prompts in the terminal by entering the required inputs step by step until the full results board is displayed.
